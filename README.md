# ADHD Focus Bot v3

## Что нового в этой версии
- Спрашивает имя и пол при первом запуске → персонализация на русском
- Утро: разминка + фокус + задачи A/B1/B2/C1/C2/C3 + free writing + благодарность + inner child
- Вечер: достижения + похвала + highlights + планы A/B/C на завтра + AI-анализ дня
- 🧠 Навык дня — ежедневный совет из тренинга навыков СДВГ (13 навыков)
- 🤖 AI-коуч с 6 быстрыми кнопками + свободный текст
- 🌅 AI-мотивация утром на основе твоего фокуса
- 🌙 AI-анализ дня вечером
- Уведомления: 9:00 и 21:00 по Тбилиси — настроены по умолчанию

## Деплой на Railway

### 1. Создай бота у @BotFather
→ /newbot → скопируй токен

### 2. Загрузи на GitHub
Создай репозиторий, загрузи `bot.py` и `requirements.txt`

### 3. Railway
- railway.app → New Project → GitHub repo
- Variables → добавь:

| Переменная | Значение | Обязательно |
|-----------|---------|------------|
| `BOT_TOKEN` | токен от BotFather | ✅ |
| `NOTIFY_USER_ID` | твой Telegram ID | ✅ для уведомлений |
| `ANTHROPIC_KEY` | sk-ant-... | ❌ (без него нет AI) |
| `MORNING_HOUR_UTC` | 5 | ❌ (9:00 Тбилиси) |
| `EVENING_HOUR_UTC` | 17 | ❌ (21:00 Тбилиси) |

### 4. Узнай свой Telegram ID
Напиши @userinfobot — он ответит числом типа `123456789`

### 5. Часовые пояса
UTC+4 (Тбилиси):
- 9:00 = MORNING_HOUR_UTC=5
- 21:00 = EVENING_HOUR_UTC=17

Для Москвы (UTC+3): 5→6, 17→18
